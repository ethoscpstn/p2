TROUBLESHOOTING
───────────────────────────────────────────────────────────
If migration fails:
→ Check database connection
→ Verify user has ALTER TABLE privileges
→ Run migration again (it's safe to re-run)

If ML prediction fails:
→ Check ngrok tunnel is running
→ Verify ML_BASE URL in config_keys.php
→ Check browser console for errors
→ Test API endpoint directly

If dark mode doesn't persist:
→ Check browser localStorage enabled
→ Clear cache and try again
→ Check darkmode.js is loaded

If forms don't submit:
→ Check CSRF token present in form
→ Verify file upload limits
→ Check browser console for JS errors
